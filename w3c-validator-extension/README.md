# Extension W3C Validator Checker

Une extension Firefox simple pour vérifier la validité W3C de la page web actuelle.

## Fonctionnalités

- Vérifie la validité W3C de la page actuelle
- Affiche le nombre d'erreurs et d'avertissements
- Lien direct vers le rapport complet du validateur W3C
- Interface simple et intuitive

## Installation

1. Téléchargez ou clonez ce dépôt
2. Ouvrez Firefox et tapez `about:debugging` dans la barre d'adresse
3. Cliquez sur "Ce Firefox" dans le menu de gauche
4. Cliquez sur "Charger un module complémentaire temporaire..."
5. Sélectionnez le fichier `manifest.json` dans le dossier de l'extension

## Utilisation

1. Cliquez sur l'icône de l'extension dans la barre d'outils de Firefox
2. Cliquez sur le bouton "Vérifier cette page"
3. Attendez les résultats de la validation
4. Cliquez sur le lien pour voir le rapport complet si nécessaire
